## Usage

### Basic Usage

#### Import the Component

```tsx
import { MorphImage } from '@/components/morph-image';
```

#### Use the Component

```tsx
<MorphImage src="/images/card-1.jpg" alt="Morph Image" className="w-[300px]" />
```

## Props

All `motion.img` element props are supported.
